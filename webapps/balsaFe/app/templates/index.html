Welcome to BALSA

Basic Assembly Language Source Assist

BALSA is a Prompt Engineering tool for 1960s era IBM Assembler. It can provide code samples, explain assembly code on an individual instruction basis, commentary for small sequences of instructions and provide insight to poorly written assembly code. In all cases, assembly refers to BAL assembly.
