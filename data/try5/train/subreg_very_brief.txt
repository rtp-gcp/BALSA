
RESULT

```
         SR   R1,R2
         ST   R1,DIFF
```

PROMPT

Write some code in BAL Assembly to subtract two fullword registers using integers and store the result in memory with a field name of DIFF.  


Assume the values to add are already in registers R1 and R2.  

R1 will contain the minuend.

R2 will contain the subtrahend.

Do not show any subroutine standard entry and exit code.

Do not show how the field name should be declared.
